# Extended UI+

[![Mindustry Version](https://img.shields.io/badge/Mindustry-v154+-orange)](https://mindustrygame.github.io/)
[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)

**Расширьте свой интерфейс полезными функциями!**

Extended UI+ — это мод для Mindustry который добавляет множество улучшений интерфейса для более комфортной игры и служит улучшением для обычного Extended UI 

---

## ✨ Особенности

### 📐 Таблица схем
- **Категории** — организуйте схемы по вкладкам
- **Предпросмотр** — смотрите схему перед использованием
- **Быстрый доступ** — используйте схемы в один клик
- **Импорт/Экспорт** — делитесь настройками с друзьями

### 🤖 Таблица юнитов
- **Отслеживание по командам** — видите лучших юнитов каждой команды
- **Прогресс производства** — следите за созданием юнитов
- **Авто-контроль** — автоматически берите контроль над нужными юнитами

### 🏭 Улучшения производства
- **Прогресс-бары** — отображаются на фабриках и реконструкторах
- **Авто-заполнение** — постройки автоматически получают ресурсы
- **Индикатор энергии** — показывается потребление и производство
- **Прирост ресурсов** — скорость производства в реальном времени

### 💪 Боевые улучшения
- **Полоски здоровья** — здоровье и щиты юнитов
- **Оповещения об атаках** — предупреждения о нападениях
- **Потеря поддержки** — следите за потерей юнитов
- **Информация о блоках** — дополнительная информация о вражеских постройках

### 🎮 Удобство игры
- **Расширенный зум** — приближайте камеру дальше обычного
- **Перетаскивание ядра** — создавайте хранилища перетаскиванием
- **Выбор области** — создавайте схемы выделением области
- **Настройки** — отключите любые функции в настройках

## 🚀 Установка

### Автоматическая установка
1. Откройте Mindustry
2. Перейдите в **Моды** → **Мастерская**
3. Найдите **Extended UI+**
4. Нажмите **Загрузить**
5. Включите мод в списке

### Ручная установка
1. Скачайте последнюю версию из [Releases](https://github.com/Salivion/Extended-UI-Plus/releases)
2. Скопируйте папку `extended-UI-v8` в `Mods/`
3. Перезапустите игру
4. Включите мод в настройках

---

## ⚙️ Настройки

Все функции можно настроить или отключить:

1. Откройте **Настройки** → **Extended UI+**
2. Выберите нужную категорию
3. Настройте параметры

### Основные настройки:
- `Показывать таблицу схем` — включить/выключить таблицу схем, переместить по X или Y
- `Показывать таблицу юнитов` — включить/выключить таблицу юнитов
- `Авто-контроль юнитов` — выбрать юнита для автоматического контроля
- `Прогресс бары фабрик` — показать прогресс производства
- `Полоски здоровья юнитов` — показать здоровье и щиты
- `Индикатор энергии` — показать статус энергосети
- `Прирост ресурсов` — показать скорость производства
- `Расширенный зум` — максимальный уровень зума (1-10)
